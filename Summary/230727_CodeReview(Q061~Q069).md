##Q066.
- 중복되지 않는 난수를 판단하는 알고리즘 구현에 다수의 팀원들이 어려움을 겪음.
- 중복검사를 위한 메소드 생성 방법 외에 아래와 같은 방법으로 문제를 해결한 case도 있었음.

        // 배열 생성
        int[] nums = new int[6];
        // 난수를 배열에 저장
        for (int i = 0; i < nums.length; i++) {
            nums[i] = (int)(Math.random() * 45) + 1;
            // 중복 검사
            for (int j = 0; j < i; j++) {
            	// 재시도를 위해 인덱스를 감소
                if (nums[i] == nums[j]) {
                    i--;
                    break;
                }
            }
        }
        // 배열 출력
        System.out.println(Arrays.toString(nums));
